# 设计规范

## 整体风格

**性冷淡风 / Dark Minimalist**

- 大量留白，克制用色
- 黑色系为主，低对比度
- 无多余装饰，线条干净
- 字体简洁统一

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | `#1a1a1a` | 深灰黑，不刺眼 |
| 卡片/单元格 | `#2a2a2a` | 次级背景 |
| 文字主色 | `#e0e0e0` | 柔白，不过亮 |
| 文字副色 | `#888` | 辅助信息 |
| 绿色（正确） | `#538d4e` | 柔和绿色 |
| 黄色（接近） | `#b59f3b` | 柔和金色 |
| 黑色（错误） | `#3a3a3a` | 深灰，非纯黑 |
| 边框 | `#444` | 低调分割 |
| 输入框背景 | `#333` | 略亮于背景 |
| 输入框聚焦 | `#555` | 聚焦指示 |

## 排版

- 字体：`system-ui, -apple-system, sans-serif`
- 标题：`font-weight: 700`, `letter-spacing: 0.05em`
- 正文：`font-weight: 400`
- 列头：`font-size: 0.7rem`, 大写, `letter-spacing: 0.1em`
- 数据单元格：`font-size: 0.85rem`, `font-weight: 600`

## 布局

- 最大宽度：`600px`
- 居中排列，两侧自动留白
- 移动端：全宽 + 16px 内边距
- 猜测表格：7 列等宽 CSS Grid
- 搜索框：全宽，圆角 8px

## 交互动效

- 猜测行出现：淡入 + 下滑（`fadeInDown`, 300ms）
- 颜色切换：`transition: background-color 0.3s ease`
- 按钮悬停：亮度提升 10%
- 自动补全下拉：展开/收起 150ms
- 箭头提示：出现在黄色单元格旁，字体略小

## 响应式断点

- 默认：`max-width: 600px` 居中
- `< 480px`：列头文字缩小，单元格 padding 缩小
- `< 375px`：列头缩写（如 Conf → CNF），最小可用

## 可访问性

- 色盲友好：颜色不是唯一区分方式（配合文字 + 箭头）
- 键盘导航：Tab → 搜索框 → 下拉列表 → 剪影按钮
- 语义 HTML：使用 `<table>`, `<thead>`, `<tbody>`
