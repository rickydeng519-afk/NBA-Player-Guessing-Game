# 开发需求文档

## 项目概述

NBA 球员猜测游戏（Poeltl 风格），用户通过 8 次猜测机会找出当日谜底球员。

## 功能需求

### F1 - 每日谜底
- 每天自动生成一个现役 NBA 球员作为谜底
- 同一自然日所有用户猜的是同一个球员
- 基于日期种子哈希选取，确保公平随机

### F2 - 球员搜索
- 输入框输入球员名，实时自动补全
- 匹配规则：大小写不敏感，子串匹配
- 最多显示 8 条建议
- 支持键盘 ↑↓ 导航 + Enter 确认 + 鼠标点击

### F3 - 属性比对（7 列）
| 列 | 含义 | 绿色 | 黄色 | 黑色 |
|-----|------|------|------|------|
| Team | 当前球队缩写 | 同队 | 曾效力过该队 | 从未效力 |
| Conf | 分区 (East/West) | 同分区 | — | 不同分区 |
| Div | 赛区 | 同赛区 | — | 不同赛区 |
| Pos | 位置 | 完全一致 | 有重叠位置 | 无重叠 |
| Ht | 身高 | 一致 | ±2 英寸 + ↑↓箭头 | 超过 2 英寸 |
| Age | 年龄 | 一致 | ±2 岁 + ↑↓箭头 | 超过 2 岁 |
| # | 球衣号码 | 一致 | ±2 + ↑↓箭头 | 超过 2 |

### F4 - 游戏流程
- 8 次猜测机会
- 猜对 → 胜利弹窗 + 分享按钮
- 8 次用完未猜对 → 失败 + 显示谜底
- 已猜过的球员不可重复猜测

### F5 - 剪影提示
- 点击按钮显示谜底球员的剪影图片
- 不计入猜测次数
- 可随时查看/隐藏

### F6 - 战绩分享
- 游戏结束后生成 emoji 方格战绩
- 一键复制到剪贴板
- 格式类似 Wordle 分享

### F7 - 状态持久化
- LocalStorage 保存当日游戏状态
- 刷新页面不丢失进度
- 第二天自动重置

## 非功能需求

- 零依赖：单个 HTML 文件，双击浏览器即可运行
- 移动端优先：响应式设计，375px 起适配
- 加载速度：首次渲染 < 1 秒
- 浏览器兼容：Chrome/Safari/Firefox 最新两个版本
